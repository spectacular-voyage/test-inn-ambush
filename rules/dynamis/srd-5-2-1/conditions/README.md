# SRD 5.2.1 Condition Rules

This package is a curated extraction from the CC-BY-4.0 SRD 5.2.1 PDF. The permanent authoring surface is `conditions.effects.csv`; `conditions.ttl` is generated from that CSV by `tools/srd-extraction/extract-srd.mjs build-rdf` in the Stagecraft repo.

Rows distinguish source-prose effects from codified affordance masks. When an effect is mechanical but not an affordance mask, or when it depends on table adjudication, the CSV says so in `mechanicalStatus` and `adjudicationNote` instead of inventing mechanics.
