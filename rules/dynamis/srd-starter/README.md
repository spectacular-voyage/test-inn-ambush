# Dynamis SRD Starter Rules

This directory is the first fixture rules package for Dynamis. It proves the data shape only; it is not a complete SRD extraction.

Files:

- `dynamis-srd-starter.ttl`: RDF rules package, action definitions, decision table rows, and attribution.
- `dynamis-srd-starter.decision-table.csv`: CSV authoring projection checked against the RDF.

The package carries CC-BY-4.0 attribution for SRD-derived starter material. Vocabulary and fixture identifiers stay generic and do not use product trademarks.
